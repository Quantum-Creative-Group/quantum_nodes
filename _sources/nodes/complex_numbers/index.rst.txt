Complex numbers
===============


Description
###########

This implementation allows you to manipulate complex numbers through
new sockets and nodes


Sockets
*******

* **Complex128 socket** - It can intake or output the complex numbers. This
  socket is an implementation of the `numpy double complex <https://numpy.org/devdocs/reference/arrays.scalars.html#numpy.complex128>`_
  data type.
